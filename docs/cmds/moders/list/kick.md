### **Команда для модераторов: исключение пользователя из беседы**
<br>

#### **Алиасы**:
`kick`, `кик`, `кикнуть`, `исключить`


#### **Примеры использования**:
- `kick @id1` (`kick <user:mention>`)
- `kick` (`kick`)
  > Ответ на сообщение пользователя


#### **Описание**:
исключает пользователя из беседы
